Le Savoir des Boucles

Automatiser les répétitions pour gagner du temps.

1. La boucle FOR

Elle sert à parcourir une série d'éléments ou à répéter une action un nombre précis de fois.

for i in range(10):
    print("Ceci est la ligne n°", i)

Note : range(10) va de 0 à 9 (soit 10 répétitions).

2. La boucle WHILE

Elle répète une action **tant que** une condition est vraie. C'est idéal pour les menus de jeux ou les systèmes de combat.

continuer = True
while continuer == True:
    reponse = input("Continuer ? o/n")
    if reponse == "n":
        continuer = False

⚠️ Loi de Python : L'indentation

Tout ce qui est à l'intérieur d'une boucle doit être décalé par **4 espaces** (ou une tabulation). Sans cela, le code ne fonctionnera pas.
