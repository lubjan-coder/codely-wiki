Le Savoir des Variables

Comprendre comment Python mémorise les informations.

# Les types de données

str (String)

Du texte entre guillemets.
pseudo = "Alex"

int (Integer)

Nombre entier sans virgule.
niveau = 5

# Opérations mathématiques

On peut modifier une variable en utilisant des opérateurs :

score = 10
score = score + 5 # Addition (vaut 15)
vie = vie - 1 # Soustraction
double = score * 2 # Multiplication
moitie = score / 2 # Division

⚠️ Règles de nommage

  • OUI mon_score (Minuscules et underscore)
  • NON 1er_joueur (Ne jamais commencer par un chiffre)
  • NON mon score (Pas d'espaces)
